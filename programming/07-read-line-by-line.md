# Read line by line

Write a program to read a txt file, line by line, and output each line to the screen.

## Sample txt file

```txt
a
girl
called
jane
went
to
the
zoo
to
see
some
lions.
She
was
the
zoo
keeper.
```

## Variants

1. Wait for 1 second before printing each line.

2. Output into another file.
